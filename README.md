# Lab_11_OS

Склад команди: Андурущак, Берещак, Гладкий

Андрущак - сервер для стгування погоди через відкриті API, розсилка погоди
Гладкий - процес-користувач, який підписується на розсилку погоди
Берещак - процес, який зв'язує між собою сервер та користувача
