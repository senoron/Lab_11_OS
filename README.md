# Lab_11_OS (i)

Склад команди: Андурущак, Берещак, Гладкий, Емірова

Андрущак - сервер для стгування погоди через відкриті API, розсилка погоди


Гладкий - процес, який зв'язує між собою сервер та користувача


Берещак - процес-користувач, який підписується на розсилку погоди

Емірова - інтерфейсна частина для програм
